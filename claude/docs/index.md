# tag-ai-literacy

Initial mkdocs for AI Literacy site


!!! info "Project Overview"
    A WordPress plugin that creates an educational resource using MkDocs. This project provides a set of markdown files (starting with glossary.md) to help clarify, educate, and elucidate **AI terminology, benefits, risks, and strategies**. An AI literacy resource aimed at making AI concepts accessible and understandable.
